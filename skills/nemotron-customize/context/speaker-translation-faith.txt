# Pipeline Overview for Translation

**Goal**: Understand how the translation pipeline works from input JSONL to translated output, and how FAITH evaluation fits in.

## Architecture

The translation pipeline has two main flows: translation and FAITH evaluation.

```{mermaid}
flowchart LR
    A[Input JSONL] --> B[Preprocess / Segment]
    B --> C[Translate]
    C --> D[Postprocess]
    D --> E[Raw + Final output]
    E --> F{FAITH?}
    F -->|Yes| G[FAITH eval]
    G --> H[faith_scored/]
    F -->|No| I[Done]
```

## Translation Flow

### Stage 1: Load and Preprocess

Load JSONL and extract text to translate.
Segment the text for the backend.

1. **Load input JSONL** file with one JSON object per line.

   Chunk the input into N chunks, according to `-n` argument.
   The pipeline processes the chunks serially.
1. For each record in a chunk, **extract the text to translate** from the specified field.
   You specify the field in the `translation.fields_to_translate`, such as `message.*.content`, from the config file.
   The asterisk wildcards indicates to expand to multiple values per record.
1. **Segment** the extracted text before translation:
   - **Coarse (default)**: Line-based, skip code blocks (```), preserve indentation.
   - **Fine**: Sentence-like units using spaCy, preserve exact separators.
     Use fine segmentation if you find truncated text in the output.

   Segmenting also stores reconstruction metadata so that postprocessing can reassemble the documents in the exact order from the translated segments.

### Stage 2: Translation

1. **Translate segments** using the specified server endpoint.

   The pipeline performs the following actions on the segments:

   - Creates a temporary file, such as `output_chunk_*.jsonl-async`, and begins writing translated records into it.
     The pipeline uses these files to resume after an interrupted run.
   - On completion of the chunk, creates an `output_chunk_*.jsonl` (also temporary, deleted when all translation completes), and `output_chunk_*_raw.jsonl` files.
   - Deletes the temporary `-async` file.
   - Creates the `output_chunk_*.jsonl.done` file.

### Stage 3: Postprocess

Postprocessing reassembles the individual translation results back into the original document structure.
Post-processing creates two output files.

1. Read the segment-level output file for the chunk.
2. Extract translated text from each line (and unwrap any `〘...〙` wrapper if present).
3. Using the stored reconstruction data, rebuild each original document: replace placeholders or concatenate segments with separators (coarse vs fine).
4. For each record, produce:
   - **Raw**: Original record + `translations` key (`target_lang`, `translation`, `segmented_translation`).
   - **Final**: Same schema as input; configured fields replaced by translated text.
5. Write `<stem>_raw.jsonl` and `<stem>.jsonl`, for example `output_chunk_0_raw.jsonl` and `output_chunk_0.jsonl`.

**Output**: Raw and final JSONL in `<data-directory>/translated/`.

## FAITH Evaluation Flow (Optional)

**Purpose**: Score translation quality per segment using an LLM (FAITH metric).

**Process**:

1. **Input**: Translation output that contains `translations.segmented_translation`, for example from `output.jsonl` or merged chunk raw files.
2. **Preprocess**: Extract segment pairs (source text, translated text) and build evaluation prompts (source/target language–agnostic).
3. **Generate**: LLM scores each segment, for example 1–5, on FAITH criteria; results collected per record.
4. **Postprocess**: Attach scores to each record and write to `<data-directory>/faith_scored/`.

**Output**: JSONL with a `faith_scores` (or configured) key per record.

# FAITH Evaluation

**Goal**: Understand what FAITH evaluation measures and how the FAITH pipeline uses translation output to produce quality scores.

## What FAITH Measures

FAITH is an LLM-based evaluation that scores how faithfully a translation preserves the meaning of the source text. The pipeline sends **source–translation segment pairs** to an LLM with a structured prompt and parses scores, for example 1–5 per category, from the model output. Results are attached to each record so you can filter or analyze by quality.

## When to Run FAITH

- After you have **translation output** (raw or final) that includes **segment-level** source and translation text.
- The pipeline expects input with a `translations` key (or your configured `translation_key`) containing **`segmented_translation`**: per-field lists of `{src, tgt}` pairs. This is exactly what the translation pipeline writes in **raw** output (`*_raw.jsonl`).
- Use FAITH to filter low-quality segments, compare backends, or monitor translation quality over time.

## How the FAITH Pipeline Runs

1. **Input**: By default, the FAITH CLI reads from `<data-directory>/translated/`. It looks for `output.jsonl`; if not found, it looks for `output_chunk_*_raw.jsonl`, merges them in order into a temporary file, and uses that. You can override with `--input-file`.
2. **Preprocess**: For each record, the pipeline reads `translations.segmented_translation` (or the equivalent for your `translation_key`). Each `{src, tgt}` pair becomes one evaluation point (source text, translated text, source/target language names for the prompt).
3. **Generate**: An LLM is called for each segment (or batch) with a FAITH scoring prompt. The prompt is language-agnostic (uses source/target language names). Model output is parsed to extract scores.
4. **Postprocess**: Scores are attached to each record, for example under `faith_scores` or your configured `scores_key`. Output is written to **`<data-directory>/faith_scored/`**.

So FAITH does **not** re-translate; it only **scores** existing translations using an LLM.

## Input Requirements

- Input must be JSONL produced by the translation pipeline (or compatible): each line must have the `translation_key` object with **`segmented_translation`** (field → list of `{src, tgt}`).
- If you only have final output (`output.jsonl` with replaced text and no `translations` key), you cannot run FAITH on it unless you reconstruct segment pairs elsewhere. Prefer raw output or merged chunk raw files.

## Output

- **Directory**: `<data-directory>/faith_scored/`.
- **Content**: JSONL with the same record structure as input, plus a scores key such as `faith_scores` containing per-segment or per-record scores and metadata, for example `source_lang`, `target_lang`, and `field_scores`.

# Translation Configuration Reference

Complete reference for the translation pipeline configuration file (`translate_config.yaml`). CLI arguments override config file values.

## Configuration File Location

Default path: `src/speaker/driver/translate/configs/translate_config.yaml`. Pass `--config <path>` to use another file.

## Top-Level Sections

| Section | Purpose |
|---------|---------|
| `model_type` | Backend: `llm`, `nmt`, `google`, or `aws`. |
| `server` | Server type, model (LLM), and address (LLM or NMT). |
| `translation` | Source/target language, fields to translate, translation key, prompt config. |
| `processing` | num_chunks, max_concurrent_requests, batch_size. |
| `inference` | LLM-only: temperature, top_k, top_p. |
| `google` | Google Cloud: project_id, location, api_version. |
| `aws` | AWS: region. |

## Parameters

```{list-table}
:header-rows: 1
:widths: 22 12 66

* - Parameter
  - Type
  - Description
* - `model_type`
  - string
  - Backend: `llm` (OpenAI-compatible), `nmt` (HTTP server), `google`, or `aws`.
* - `server.type`
  - string
  - For LLM: typically `openai`.
* - `server.model`
  - string
  - LLM model name, for example `mistralai/mistral-small-3.1-24b-instruct-2503`. Used only when `model_type: llm`.
* - `server.address`
  - string
  - Base URL: LLM OpenAI-compatible endpoint or NMT server, for example `http://localhost:8000`.
* - `translation.source_lang`
  - string
  - Source language code (ISO 639-1), for example `en`.
* - `translation.target_lang`
  - string
  - Target language code, for example `hi` or `zh`.
* - `translation.fields_to_translate`
  - list
  - Field paths; `*` wildcard supported, for example `messages.*.content`.
* - `translation.translation_key`
  - string
  - Key under which translations are stored in raw output (default: `translations`).
* - `translation.prompt_config`
  - string
  - Path to prompt YAML for LLM translation (used when `model_type: llm`).
* - `processing.num_chunks`
  - integer
  - Number of chunks for processing sequentially (default: 1).
* - `processing.max_concurrent_requests`
  - integer
  - Max in-flight requests per chunk. Default in sample config: 16; code default if key omitted: 1.
* - `processing.batch_size`
  - integer
  - Texts per request; NMT uses batching; LLM typically 1.
* - `inference.temperature`
  - float
  - LLM sampling temperature (0.0 = greedy).
* - `inference.top_k`
  - integer
  - Top-k sampling (-1 = disabled).
* - `inference.top_p`
  - float
  - Top-p (nucleus) sampling.
* - `google.project_id`
  - string
  - Google Cloud project ID (required for v3 API).
* - `google.location`
  - string
  - Google Cloud location, for example `global`.
* - `google.api_version`
  - string
  - `v2` or `v3`.
* - `aws.region`
  - string
  - AWS region for Translate, for example `us-east-2`.
```

## Sample Configuration

```yaml
model_type: "llm"

server:
  type: "openai"

  project_id: "my-project"
  location: "global"
  api_version: "v2"

aws:
  region: "us-east-2"
```

# Input and Output Reference

Reference for the translation pipeline: input JSONL schema, output directory layout, raw output (with translation metadata), and final output.

## Input JSONL Files

- One JSON object per line.
- Field paths in `translation.fields_to_translate`, such as `messages.*.content`, specifies the string values to translate.
  When the field path includes the `*` wildcard, the translation output field becomes a list.

The following JSON shows the format that the pipeline can process by default, `messages.*.content`:

```json
{
  "uuid": "73ffd37c-f505-41a0-ab12-6b18ae8faae1",
  "messages": [
    { "role": "user", "content": "Hello, how are you today?", "tool_calls": [] },
    { "role": "assistant", "content": "I am doing well, thank you.", "tool_calls": [] }
  ],
  "metadata": "{\"conversation_id\": \"...\"}"
}
```

## Output Directory

You specify the `--data-directory` argument.
The pipeline stores all translation output under`<data-directory>/translated/`.
The `translated/` portion of the path is not configurable.

### Output Directory Layout

For a single-chunk run with `num_chunks=1`, the output directory resembles the following:

```text
<data-directory>/translated/
├── output.jsonl                   # Final merged output
├── output.jsonl.done              # Completion marker for final output
├── output_chunk_0.jsonl.done      # Chunk 0 completion marker
└── output_chunk_0_raw.jsonl       # Chunk 0 raw output
```

During the translation process, the pipeline creates temporary files, such as `output_chunk_0.jsonl-async`.
The pipeline uses these files to resume after an interrupted run.
After the pipeline processes a chunk, it removes the temporary file, writes the raw file, and creates the `.done` completion marker file.

For a chunked run, such as `-n 4`, the output directory resembles the following:

```text
<data-directory>/translated/
├── output.jsonl                   # Final merged output
├── output.jsonl.done              # Completion marker for final output
├── output_chunk_0_raw.jsonl       # Chunk 0 raw output
├── output_chunk_0.jsonl.done      # Chunk 0 completion marker
├── output_chunk_1_raw.jsonl       # Chunk 1 raw output
├── output_chunk_1.jsonl.done      # Chunk 1 completion marker
├── output_chunk_2_raw.jsonl       # Chunk 2 raw output
├── output_chunk_2.jsonl.done      # Chunk 2 completion marker
├── output_chunk_3_raw.jsonl       # Chunk 3 raw output
└── output_chunk_3.jsonl.done      # Chunk 3 completion marker
```

The pipeline divides the input into chunks and processes the chunks serially, creating numbered raw chunk files.

When the pipeline completes translation, it merges all `output_chunk_*.jsonl` files to produce `output.jsonl`.
The pipeline also creates an `output.jsonl.done` file to indicate that translation completed successfully.

### Raw Format Output Files

The `*_raw.jsonl` files contain the original record, plus a `translations` key, or the key you specify for `translation_key`.

:::{list-table}
:header-rows: 1
:widths: 25 75

* - Key
  - Description
* - (all original keys)
  - Unchanged.
* - `translations`
  - Object with: `target_lang`, `translation` (field → translated text or list), `segmented_translation` (field → list of `{original, translated}` pairs).
:::

Use case: Debugging, quality inspection, alignment verification.
The FAITH evaluation pipeline uses the raw files to assess the quality of translation from the original content.

```json
{
  "uuid": "73ffd37c-f505-41a0-ab12-6b18ae8faae1",
  "messages": [
    { "role": "user", "content": "Hello, how are you today?", "tool_calls": [] },
    { "role": "assistant", "content": "I am doing well, thank you.", "tool_calls": [] }
  ],
  "metadata": "...",
  "translations": {
    "target_lang": "hi",
    "translation": { "content": ["नमस्ते, आज आप कैसे हैं?", "मैं अच्छा हूं, धन्यवाद।"] },
    "segmented_translation": {
      "content": [
        { "src": "Hello, how are you today?", "tgt": "नमस्ते, आज आप कैसे हैं?" },
        { "src": "I am doing well, thank you.", "tgt": "मैं अच्छा हूं, धन्यवाद।" }
      ]
    }
  }
}
```

### Final Output Format

The `output.jsonl` file is the final output for production use.
It uses the same schema as input.
Configured fields are replaced by translated text.

```json
{
  "uuid": "73ffd37c-f505-41a0-ab18ae8faae1",
  "messages": [
    { "role": "user", "content": "नमस्ते, आज आप कैसे हैं?", "tool_calls": [] },
    { "role": "assistant", "content": "मैं अच्छा हूं, धन्यवाद।", "tool_calls": [] }
  ],
  "metadata": "..."
}
```

### Output File Comparison

:::{list-table}
:header-rows: 1
:widths: 20 40 40

* - Feature
  - Raw (``*_raw.jsonl``)
  - Final (``output.jsonl``)
* - Original text
  - Preserved in original fields
  - Replaced with translation
* - Translations
  - In separate ``translations`` key
  - Moved to original message fields
* - Metadata
  - Includes ``segmented_translation``
  - Not included
* - Schema
  - Input schema + ``translations``
  - Same as input
* - Use case
  - Debugging, QA, FAITH input
  - Production, downstream
:::

## FAITH Evaluation Output

The `faith_scored` portion of the path is not configurable.

```text

├── output.jsonl.done
└── output_chunk_0.jsonl.done
```

def load_config(config_path: str) -> dict:
    """Load configuration from YAML file."""
    with open(config_path, 'r') as f:

def main():
    """Main entry point for translation pipeline."""

    parser = argparse.ArgumentParser(
        description="Translate data using the translation pipeline"
    )
    parser.add_argument(
        "-c",

def _get_spacy_nlp(src_lang: str = "en"):
    """
    Lazy load spaCy model for NMT sentence splitting.
    
    Args:
        src_lang: ISO 639-1 language code (e.g., 'en', 'de', 'hi')
    
    Returns:
        Loaded spaCy model appropriate for the source language.
    """
    model_name = SPACY_LANG_MODELS.get(src_lang, SPACY_FALLBACK_MODEL)
    
    if model_name not in _nlp_cache:
        import spacy
        try:
            nlp = spacy.load(model_name)
        except OSError:
            LOG.warning(f"spaCy model '{model_name}' not found for lang '{src_lang}', using fallback '{SPACY_FALLBACK_MODEL}'")

            nlp = spacy.load(model_name)
        
        nlp.max_length = 10_000_000  # Handle very long texts (10M chars)
        _nlp_cache[model_name] = nlp
        LOG.info(f"spaCy model '{model_name}' loaded for lang '{src_lang}' with max_length: {nlp.max_length}")
    
    return _nlp_cache[model_name]


def split_into_sentences_with_structure(text: str, src_lang: str = "en") -> List[Tuple[str, str]]:
    """
    Split text using spaCy, then apply custom regex patterns while preserving exact structure.
    Returns list of tuples: (sentence_text, separator_after)
    
    Args:
        text: The text to split into sentences.
        src_lang: ISO 639-1 language code (e.g., 'en', 'de', 'hi') for loading
                  the appropriate spaCy model.
    
    Returns:
        List of tuples where each tuple is (sentence_text, separator_after).
        This ensures that when we rejoin: text_unit + separator for each tuple,
        we get back the exact original text.
    """
    nlp = _get_spacy_nlp(src_lang)
    
    # Custom regex pattern for special characters that should be treated as separators
    special_separator_pattern = r"(\#{2,}|\_{2,}|\…{2,}|\%{2,}|\+{2,}|\.{2,}|\-{3,}|\*{2,}|\~{2,}|\={2,}|\!{2,}|\n|\t|\‣|\⁃|\⁌|\⁍|\●|\○|\•|\·|\◘|\◦|\⦾|\⦿|\|)"

    doc = nlp(text)
    spacy_sentences = list(doc.sents)

    spacy_units_with_separators = []
    
    # Check if there's text before the first spaCy sentence
    if spacy_sentences:
        first_sent_start = spacy_sentences[0].start_char
        if first_sent_start > 0:
            leading_text = text[:first_sent_start]
            spacy_units_with_separators.append(("", leading_text))
    
    for i, sent in enumerate(spacy_sentences):
        sent_start = sent.start_char
        sent_end = sent.end_char
        
        # Calculate the separator after this sentence
        if i < len(spacy_sentences) - 1:
            next_sent_start = spacy_sentences[i + 1].start_char
            separator = text[sent_end:next_sent_start]
        else:
            separator = text[sent_end:]
        
        sentence_text = text[sent_start:sent_end]
        spacy_units_with_separators.append((sentence_text, separator))

    # Process each spaCy unit for special separators
    all_text_units = []
    
    for sent_text, sent_separator in spacy_units_with_separators:
        special_matches = list(re.finditer(special_separator_pattern, sent_text))
        
        if not special_matches:
            # Strip leading/trailing spaces from text unit and add to separator
            stripped_text = sent_text.strip()
            leading_spaces = sent_text[:len(sent_text) - len(sent_text.lstrip())]
            trailing_spaces = sent_text[len(sent_text.rstrip()):]
            
            new_separator = trailing_spaces + sent_separator
            
            if leading_spaces and stripped_text:
                all_text_units.append(("", leading_spaces))
            
            all_text_units.append((stripped_text, new_separator))
        else:
            # Split this sentence by special separators
            last_end = 0
            for match in special_matches:
                split_start = match.start()
                split_end = match.end()
                
                text_unit = sent_text[last_end:split_start]
                separator = sent_text[split_start:split_end]
                
                stripped_text_unit = text_unit.strip()
                leading_spaces = text_unit[:len(text_unit) - len(text_unit.lstrip())]
                trailing_spaces = text_unit[len(text_unit.rstrip()):]
                
                new_separator = trailing_spaces + separator
                
                if leading_spaces and stripped_text_unit:
                    all_text_units.append(("", leading_spaces))
                
                all_text_units.append((stripped_text_unit, new_separator))
                last_end = split_end
            
            # Handle remaining text after last special separator
            if last_end < len(sent_text):
                remaining = sent_text[last_end:]
                stripped_remaining = remaining.strip()
                leading_spaces = remaining[:len(remaining) - len(remaining.lstrip())]
                trailing_spaces = remaining[len(remaining.rstrip()):]
                
                new_sent_separator = trailing_spaces + sent_separator
                
                if leading_spaces and stripped_remaining:
                    all_text_units.append(("", leading_spaces))
                
                all_text_units.append((stripped_remaining, new_sent_separator))
            else:
                if sent_separator:
                    all_text_units.append(("", sent_separator))

    # Verify reconstruction
    reconstructed = "".join(text_unit + separator for text_unit, separator in all_text_units)
    if text != reconstructed:
        LOG.warning("Structure mismatch in sentence splitting, falling back to single unit")
        return [(text, "")]
    
    return all_text_units


def is_line_translatable_content(line: str) -> bool:
    """Determines if the content of a line is considered translatable."""
    stripped_line = line.strip()
    if not any(char.isalpha() for char in stripped_line):

    if stripped_line.startswith("<") and stripped_line.endswith(">"):
        return False
    return True


def _get_all_nested_fields(obj, path: str) -> List[str]:
    """A helper function that finds all string values for a field specified by `path`."""
    parts = path.split(".")

    def finder(current_obj, current_path, collected_values):
        if not current_path:
            return

        key = current_path[0]
        rest_of_path = current_path[1:]

        if key == "*":
            if isinstance(current_obj, list):
                for item in current_obj:
                    if not rest_of_path:
                        if isinstance(item, str):
                            collected_values.append(item)
                    else:
                        finder(item, rest_of_path, collected_values)
            return

        if isinstance(current_obj, dict) and key in current_obj:
            if not rest_of_path:
                value = current_obj[key]
                if isinstance(value, str):
                    collected_values.append(value)
            else:
                finder(current_obj[key], rest_of_path, collected_values)

    found_values = []
    finder(obj, parts, found_values)
    return found_values


def _set_all_nested_fields(obj, path: str, translated_values: List[str]) -> None:
    """A helper function that sets all string values for a field specified by `path` with translated values."""
    parts = path.split(".")
    value_index = [0]  # Using list to maintain reference across recursive calls

    def setter(current_obj, current_path):
        if not current_path:
            return

        key = current_path[0]
        rest_of_path = current_path[1:]

        if key == "*":
            if isinstance(current_obj, list):
                for item in current_obj:
                    if not rest_of_path:
                        # This shouldn't happen - we're looking for nested string fields
                        pass
                    else:
                        setter(item, rest_of_path)
            return

        if isinstance(current_obj, dict) and key in current_obj:
            if not rest_of_path:
                # This is the field we want to replace
                if isinstance(current_obj[key], str) and value_index[0] < len(translated_values):
                    current_obj[key] = translated_values[value_index[0]]
                    value_index[0] += 1
            else:
                setter(current_obj[key], rest_of_path)

    setter(obj, parts)

    """

        lang_code: ISO 639-1 (2-letter) language code

    Returns:
        Full English name of the language, or original code if not found
    """

    lang_code = lang_code.lower().strip()
    language = iso639.Lang(lang_code)
    if language:
        return language.name

    return lang_code


@nested_dataclass(kw_only=True)
class TranslationConfig(GenerateSolutionsConfig):
    source_lang: str = "en"
    target_lang: str = "zh"
    use_skipme: bool = False
    fields_to_translate: List[str] = field(default_factory=lambda: ["conversations.*.value"])
    translation_key: str = "translations"
    model_type: str = "llm"  # "llm", "nmt", "google", "aws"

    # Controls how translation inputs are segmented before sending to the backend.
    # - coarse: fewer, larger segments (default)
    # - fine: more, smaller segments (useful for small-context models/backends)
    segmentation: str = "coarse"  # "coarse" | "fine"
    
    # Batch size: texts per HTTP request
    # Total texts in flight = batch_size * max_concurrent_requests
    batch_size: int = 1
    
    google: dict = field(default_factory=dict)
    aws: dict = field(default_factory=dict)

    # Override inference defaults for translation tasks to match with ported settings
    inference: InferenceConfig = field(
        default_factory=lambda: InferenceConfig(
            top_k=-1,
            temperature=0.7,
            top_p=0.8,
        )

class TranslationTask(GenerationTask):
    
    def __init__(self, cfg: TranslationConfig):
        self._reconstruction_data = None
        self._is_llm_mode = cfg.model_type.lower() == "llm"
        
        if self._is_llm_mode:
            # LLM mode: Full GenerationTask initialization (LLM, tokenizer, etc.)
            super().__init__(cfg)
            self._backend = None  # Will use inherited async_loop directly
        else:
            # Non-LLM modes (NMT, Google, AWS): Minimal setup, use backend
            self.cfg = cfg
            self.cfg.inference.extra_body = dict(self.cfg.inference.extra_body)
            
            # Create backend using factory
            self._backend = get_backend(cfg.model_type, cfg)
        
        LOG.info(f"TranslationTask initialized with {cfg.model_type} backend")
    
    def generate(self):
        """
        Main entry point - orchestrates preprocessing, translation, and postprocessing.
        
        Uses inherited load_data() and skip_completed_samples() from GenerationTask.
        """
        try:
            # Create output directory
            Path(self.cfg.output_file).absolute().parent.mkdir(parents=True, exist_ok=True)
            
            # Load and filter data (inherited from GenerationTask)
            data = self.load_data()
            
            if len(data) == 0:
                LOG.info("No data to process, skipping generation")
            else:
                # Preprocess (shared across all backends)
                data = self.preprocess_data(data)

                data = self.skip_completed_samples(data)
                
                if self._is_llm_mode:
                    # LLM: Use inherited GenerationTask flow
                    self.log_example_prompt(data)
                    
                    if self.cfg.dry_run:
                        LOG.info("Exiting without running generation as dry_run flag is set.")
                        return
                    
                    if not self.cfg.skip_filled:
                        for output_path in [Path(self.cfg.output_file), Path(self.cfg.output_file + "-async")]:
                            if output_path.exists():
                                output_path.unlink()
                    
                    self.wait_for_server()
                    asyncio.run(self.async_loop(data))  
                else:
                    # Non-LLM: Delegate to backend
                    self._backend.translate(data)
            
            # Postprocess (shared across all backends)
            self.postprocess()
        finally:
            # Ensure resources are released for non-LLM backends (e.g. aiohttp session in NMTModel)
            if not self._is_llm_mode and getattr(self, "_backend", None) is not None:
                try:
                    self._backend.close()
                except Exception as e:
                    LOG.warning(f"Failed to close translation backend cleanly: {e}")

    def preprocess_data(self, data):
        """
        Transform loaded JSONL data for translation.
        
        Args:
            data: List of original data points from load_data()

        Returns:
            List of translation data points with only the core fields:
            [{'source_lang': 'English', 'target_lang': 'Hebrew', 'src': 'text'}, ...]
        """
        LOG.info(f"Preprocessing {len(data)} data points for translation (mode: {self.cfg.model_type})")

        # Filter out skipme entries
        original_dps = [dp for dp in data if not (self.cfg.use_skipme and "_skipme" in dp and dp["_skipme"] != 0)]
        LOG.info(f"After filtering skipme: {len(original_dps)} data points")

        # Extract all text content that needs translation
        all_docs = []
        dp_info_for_translation = []

        for dp in original_dps:
            current_dp_info = []
            for field_path in self.cfg.fields_to_translate:
                has_wildcard = "*" in field_path
                path_parts = field_path.split(".")
                field_key = path_parts[-1]  # Always use the last part (e.g., "value" from "conversations.*.value")
                texts = _get_all_nested_fields(dp, field_path)
                for text in texts:
                    current_dp_info.append((field_key, text, has_wildcard))
                    all_docs.append(text)
            dp_info_for_translation.append(current_dp_info)

        # Default is coarse segmentation.
        if (self.cfg.segmentation or "coarse") == "fine":
            return self._preprocess_fine(all_docs, original_dps, dp_info_for_translation)
        return self._preprocess_coarse(all_docs, original_dps, dp_info_for_translation)

    def _preprocess_coarse(self, all_docs, original_dps, dp_info_for_translation):
        """
        Coarse segmentation: split text into relatively larger units (lines), with code block handling.
        
        - Splits document by newlines
        - Skips code blocks (``` ... ```)
        - Preserves leading whitespace for each line
        - Used for LLMs with large context windows
        """
        # Before we translate, we need to extract all the lines that require translation from the documents.
        # However, it's not enough to just extract the translatable lines: we also need to remember the structure and formatting of the original documents,
        # so that after translation, we can reconstruct the documents with the translated lines in their correct places, preserving code blocks, indentation, and other formatting.
        #
        # The following variables are used to store this information:
        # - all_translatable_lines: a flat list of all lines that need translation, across all documents, in order.
        # - doc_boundaries: indices marking where each document's translatable lines start/end in all_translatable_lines.
        # - doc_templates: for each non-translatable line, we store the original line (they will be carried over as-is to the output), otherwise, we store None as placeholder.
        # - doc_translatable_indices: for each document, the indices of lines (within the document) that are to be translated.
        # - doc_leading_spaces_lists: this is an extension of doc_templates -- this stores the leading whitespace for each translatable line, so we can restore indentation.
        # - doc_original_stripped_lines_for_segmentation: for each document, the original stripped lines to help with segmentation or reference.
        #
        # All of this information will be used in the postprocess function to reconstruct the original documents with translated content, preserving formatting.

        all_translatable_lines = []
        doc_boundaries = [0]
        doc_templates = []
        doc_translatable_indices = []
        doc_leading_spaces_lists = []
        doc_original_stripped_lines_for_segmentation = []

        for doc_content in all_docs:
            lines = doc_content.split("\n")
            current_doc_template = []
            current_doc_translatable_indices = []
            current_doc_leading_spaces_list = []
            current_doc_original_stripped_lines = []
            in_code_block = False

            for i, line in enumerate(lines):
                if line.strip().startswith("```"):
                    in_code_block = not in_code_block
                    current_doc_template.append(line)
                    continue

                if in_code_block or not is_line_translatable_content(line):
                    current_doc_template.append(line)
                else:
                    num_leading_spaces = len(line) - len(line.lstrip())
                    leading_spaces = line[:num_leading_spaces]
                    stripped_line = line[num_leading_spaces:]

                    current_doc_template.append(None)
                    all_translatable_lines.append(stripped_line)
                    current_doc_translatable_indices.append(i)
                    current_doc_leading_spaces_list.append(leading_spaces)
                    current_doc_original_stripped_lines.append(stripped_line)

            doc_templates.append(current_doc_template)
            doc_translatable_indices.append(current_doc_translatable_indices)
            doc_leading_spaces_lists.append(current_doc_leading_spaces_list)
            doc_boundaries.append(len(all_translatable_lines))
            doc_original_stripped_lines_for_segmentation.append(current_doc_original_stripped_lines)

        # Store reconstruction data for postprocessing
        self._reconstruction_data = {
            "mode": "coarse",
            "original_dps": original_dps,
            "dp_info_for_translation": dp_info_for_translation,
            "doc_boundaries": doc_boundaries,
            "doc_templates": doc_templates,
            "doc_translatable_indices": doc_translatable_indices,
            "doc_leading_spaces_lists": doc_leading_spaces_lists,
            "doc_original_stripped_lines_for_segmentation": doc_original_stripped_lines_for_segmentation,
        }

        # Create translation data points
        translation_data_points = []
        for idx, text in enumerate(all_translatable_lines):
            translation_data_points.append({
                "source_lang": full_language_name(self.cfg.source_lang),
                "target_lang": full_language_name(self.cfg.target_lang),
                "src": text,
                self.cfg.async_position_key: idx,
            })

        LOG.info(f"Coarse segmentation: Created {len(translation_data_points)} translation tasks")
        return translation_data_points

    def _preprocess_fine(self, all_docs, original_dps, dp_info_for_translation):
        """
        Fine segmentation: split text into smaller units (sentence-like), with exact-structure preservation.
        
        - Uses spaCy for sentence boundary detection
        - Preserves exact separators (spaces, newlines, special chars) between sentences
        - Guarantees exact reconstruction when rejoining translated sentences
        - Used for models with limited context windows (NMT, etc.)
        """

        # The following variables are used to store this information:
        # - all_translatable_units: flat list of (text, separator) tuples that need translation.
        # - doc_sentence_data: per-doc list of units with {translatable, original, separator} for reconstruction.
        #
        # Each unit stores its separator so postprocess can rejoin: translated_text + separator.
        # This guarantees exact structure preservation (spaces, newlines, special chars).

        all_translatable_units = []  # List of (text, separator) tuples
        doc_sentence_data = []  # Store per-doc sentence info for reconstruction

        for doc_content in all_docs:
            # Use spaCy to split into sentences with structure preservation
            sentence_units = split_into_sentences_with_structure(doc_content, self.cfg.source_lang) 
            
            # Filter: only translate non-empty text units
            doc_units = []
            for text_unit, separator in sentence_units:
                if text_unit.strip() and is_line_translatable_content(text_unit):
                    # Translatable unit
                    all_translatable_units.append((text_unit, separator))
                    doc_units.append({"translatable": True, "original": text_unit, "separator": separator})
                else:
                    # Non-translatable (empty or special chars only) - keep as-is
                    doc_units.append({"translatable": False, "original": text_unit, "separator": separator})
            
            doc_sentence_data.append(doc_units)

        # Store reconstruction data for postprocessing
        self._reconstruction_data = {
            "mode": "fine",
            "original_dps": original_dps,
            "dp_info_for_translation": dp_info_for_translation,
            "doc_sentence_data": doc_sentence_data,
        }

        # Create translation data points (only translatable text units)
        translation_data_points = []
        for idx, (text_unit, separator) in enumerate(all_translatable_units):
            translation_data_points.append({
                "source_lang": full_language_name(self.cfg.source_lang),
                "target_lang": full_language_name(self.cfg.target_lang),
                "src": text_unit,
                self.cfg.async_position_key: idx,
            })

        LOG.info(f"Fine segmentation: Split into {len(translation_data_points)} translatable units")
        return translation_data_points

    def log_example_prompt(self, data):
        """Override log_example_prompt to work with translation data format."""
        if data:
            # Use the first translation data point for logging
            sample_data_point = (
                data[0]
                if isinstance(data[0], dict) and "src" in data[0]
                else {
                    "source_lang": full_language_name(self.cfg.source_lang),
                    "target_lang": full_language_name(self.cfg.target_lang),
                    "src": "Sample text to translate",
                }
            )
            filled_prompt = self.fill_prompt(sample_data_point, data)
            LOG.info("Example prompt:\nData dictionary: %s\nPrompt: %s", sample_data_point, filled_prompt)
        else:
            LOG.info("No data available for example prompt")

    def unwrap_translation_results(self, text):
        """
        Unwrap the translation results from the generated output. The format defined here is 〘...〙.

        If the result cannot be extracted, return the original text.
        """
        left_loc = text.rfind("〘")
        right_loc = text.rfind("〙")
        if left_loc != -1 and right_loc != -1 and left_loc < right_loc:
            text = text[left_loc + 1 : right_loc]
        elif left_loc != -1:
            text = text[left_loc + 1 :]
        else:
            pass  # do nothing

        return text

    def postprocess(self):
        """
        Reassemble individual translation results back into original document structure.
        Reads the output file written by base class, reconstructs documents, and writes back.
        Creates two output files:
        1. output_raw.jsonl - Contains translations as a separate key
        2. output.jsonl - Original structure with English text replaced by translations
        """
        LOG.info("Starting postprocessing: reassembling translations into original document structure")

        # Read the individual translation results written by base class
        output_file = Path(self.cfg.output_file)
        if not output_file.exists():
            LOG.warning(f"Output file {output_file} does not exist, skipping postprocessing")
            return

        # Load individual translation results
        individual_results = []
        with open(output_file, "r", encoding="utf-8") as f:
            for line in f:
                if line.strip():
                    individual_results.append(json.loads(line))

        LOG.info(f"Loaded {len(individual_results)} individual translation results")

        # Extract translated texts from results
        translated_segments_for_all_docs = []
        for result in individual_results:
            # The base class should have stored the generation in 'generation' field
            if "generation" in result:
                translated_segments_for_all_docs.append(result["generation"])
            elif "response" in result:
                translated_segments_for_all_docs.append(result["response"])
            else:
                # Fallback: look for any text field
                translated_segments_for_all_docs.append(str(result))

        reconstruction_data = self._reconstruction_data
        mode = reconstruction_data.get("mode", "coarse")

        if mode == "coarse":
            all_results, all_segmented_translations = self._postprocess_coarse(
                translated_segments_for_all_docs, reconstruction_data
            )
        else:
            all_results, all_segmented_translations = self._postprocess_fine(
                translated_segments_for_all_docs, reconstruction_data
            )

        # Build final output with translations as separate key
        original_dps = reconstruction_data["original_dps"]
        dp_info_for_translation = reconstruction_data["dp_info_for_translation"]

        final_output_data_raw = []
        final_output_data_replaced = []
        translation_idx = 0

        # Calculate expected number of documents from dp_info_for_translation
        expected_docs = sum(len(dp_info) for dp_info in dp_info_for_translation)
        if len(all_results) == expected_docs:
            for i, dp in enumerate(original_dps):
                output_dp_raw = dp.copy()
                output_dp_replaced = json.loads(json.dumps(dp))  # Deep copy
                dp_info = dp_info_for_translation[i]
                if not dp_info:
                    final_output_data_raw.append(output_dp_raw)
                    final_output_data_replaced.append(output_dp_replaced)
                    continue

                dp_translations_map, dp_segmented_map = {}, {}
                current_translation_idx = translation_idx
                
                for field_key, _, has_wildcard in dp_info:
                    if translation_idx < len(all_results):
                        translated_text = all_results[translation_idx]
                        segmented_list = all_segmented_translations[translation_idx]
                        translation_idx += 1

                        if has_wildcard:
                            dp_translations_map.setdefault(field_key, []).append(translated_text)
                            dp_segmented_map.setdefault(field_key, []).extend(segmented_list)
                        else:
                            dp_translations_map[field_key] = translated_text
                            dp_segmented_map[field_key] = segmented_list

                # Create raw output with translations as separate key
                output_dp_raw[self.cfg.translation_key] = {
                    "target_lang": self.cfg.target_lang,
                    "translation": dp_translations_map,
                    "segmented_translation": dp_segmented_map,
                }
                final_output_data_raw.append(output_dp_raw)

                # Create replaced output by setting translated values directly in the original fields
                translation_idx_reset = current_translation_idx
                for field_path in self.cfg.fields_to_translate:
                    # Get all translated values for this field path
                    translated_values = []
                    field_key = field_path.split(".")[-1]
                    has_wildcard = "*" in field_path
                    
                    # Collect translated values for this field path
                    for fk, _, hw in dp_info:
                        if fk == field_key and hw == has_wildcard:
                            if translation_idx_reset < len(all_results):
                                translated_values.append(all_results[translation_idx_reset])
                                translation_idx_reset += 1
                    
                    # Set the translated values back into the original structure
                    if translated_values:
                        _set_all_nested_fields(output_dp_replaced, field_path, translated_values)
                
                final_output_data_replaced.append(output_dp_replaced)
        else:
            LOG.error(f"The input has {expected_docs} documents, but the output has {len(all_results)} documents.")
            final_output_data_raw = original_dps
            final_output_data_replaced = original_dps

        # Write raw output (with translations as separate key)
        output_file_raw = output_file.parent / f"{output_file.stem}_raw{output_file.suffix}"
        LOG.info(f"Writing {len(final_output_data_raw)} documents with translation key to {output_file_raw}")
        with open(output_file_raw, "w", encoding="utf-8") as f:
            for dp in final_output_data_raw:
                f.write(json.dumps(dp, ensure_ascii=False) + "\n")

        # Write replaced output (original structure with translations replacing original text)
        LOG.info(f"Writing {len(final_output_data_replaced)} documents with replaced translations to {output_file}")
        with open(output_file, "w", encoding="utf-8") as f:
            for dp in final_output_data_replaced:
                f.write(json.dumps(dp, ensure_ascii=False) + "\n")

        LOG.info("Postprocessing completed successfully")

    def _postprocess_coarse(self, translated_segments_for_all_docs, reconstruction_data):
        """
        Coarse segmentation postprocessing: reconstructs documents using line templates.
        
        - Replaces None placeholders in template with translated lines
        - Restores original leading whitespace
        - Joins lines with newlines
        """
        doc_boundaries = reconstruction_data["doc_boundaries"]
        doc_templates = reconstruction_data["doc_templates"]
        doc_translatable_indices = reconstruction_data["doc_translatable_indices"]
        doc_leading_spaces_lists = reconstruction_data["doc_leading_spaces_lists"]
        doc_original_stripped_lines_for_segmentation = reconstruction_data[
            "doc_original_stripped_lines_for_segmentation"
        ]

        # Reassemble documents
        all_results = []
        all_segmented_translations = []

        for doc_idx in range(len(doc_templates)):
            template = doc_templates[doc_idx].copy()  # Make a copy to avoid modifying original
            translatable_indices = doc_translatable_indices[doc_idx]
            leading_spaces_list = doc_leading_spaces_lists[doc_idx]
            original_stripped_lines_for_doc = doc_original_stripped_lines_for_segmentation[doc_idx]

            start_offset = doc_boundaries[doc_idx]
            end_offset = doc_boundaries[doc_idx + 1]
            doc_translated_segments = translated_segments_for_all_docs[start_offset:end_offset]
            current_doc_segmented_translations = []

            if not translatable_indices:
                all_results.append("\n".join(template))
                all_segmented_translations.append([])
                continue

            for i, original_stripped_line in enumerate(original_stripped_lines_for_doc):
                idx_in_template = translatable_indices[i]
                leading_spaces = leading_spaces_list[i]
                translated_segment = self.unwrap_translation_results(doc_translated_segments[i]).lstrip()
                current_doc_segmented_translations.append({"src": original_stripped_line, "tgt": translated_segment})
                template[idx_in_template] = leading_spaces + translated_segment

            all_results.append("\n".join(template))
            all_segmented_translations.append(current_doc_segmented_translations)

        return all_results, all_segmented_translations

    def _postprocess_fine(self, translated_segments_for_all_docs, reconstruction_data):
        """
        Fine segmentation postprocessing: reconstructs documents using stored separators.
        
        - Appends original separator after each translated sentence
        - Non-translatable units kept as-is with their separators
        - Guarantees exact structure preservation (spaces, newlines, etc.)
        """
        doc_sentence_data = reconstruction_data["doc_sentence_data"]

        all_results = []
        all_segmented_translations = []
        trans_idx = 0

        for doc_units in doc_sentence_data:
            reconstructed_parts = []
            current_doc_segmented_translations = []

            for unit in doc_units:
                if unit["translatable"]:
                    # Get translation for this unit
                    if trans_idx < len(translated_segments_for_all_docs):
                        translated_segment = self.unwrap_translation_results(translated_segments_for_all_docs[trans_idx]).lstrip()
                        trans_idx += 1
                    else:
                        LOG.error(f"No translation found for unit: {unit['original']}")
                        raise RuntimeError(
                            "Postprocess failed: missing translation segment for a translatable unit. "
                            "This indicates an upstream translation failure or a mismatch between "
                            "preprocess/postprocess segmentation and produced outputs."
                        )
                    
                    current_doc_segmented_translations.append({
                        "src": unit["original"],
                        "tgt": translated_segment
                    })
                    reconstructed_parts.append(translated_segment + unit["separator"])
                else:
                    # Non-translatable: keep original
                    reconstructed_parts.append(unit["original"] + unit["separator"])

            all_results.append("".join(reconstructed_parts))
            all_segmented_translations.append(current_doc_segmented_translations)

        # Validate that all translations were consumed
        if trans_idx != len(translated_segments_for_all_docs):
            LOG.error(
                f"Translation count mismatch: processed {trans_idx} translations but received {len(translated_segments_for_all_docs)}. "
                f"Extra translations: {len(translated_segments_for_all_docs) - trans_idx}"
            )
            raise RuntimeError(
                "Postprocess failed: mismatch between number of translatable units and translation results. "
                "This indicates an upstream translation failure or a mismatch between "
                "preprocess/postprocess segmentation and produced outputs."
            )

        return all_results, all_segmented_translations

        super().__init__(cfg)
        self.cfg = cfg

    def preprocess_data(self, data):
        """
        Transform loaded JSONL data to extract original and translated segment pairs for FAITH evaluation.
        Uses segmented_translation to break down long conversations into individual segments.

        Args:
            data: List of data points with translations already present

        Returns:
            List of evaluation data points for each segment:
            [{'source_text': '...', 'translated_text': '...', 'source_language': '...', 'target_language': '...'}, ...]
        """
        LOG.info(f"Preprocessing {len(data)} data points for FAITH evaluation")

        # Filter out skipme entries
        original_dps = [dp for dp in data if not (self.cfg.use_skipme and "_skipme" in dp and dp["_skipme"] != 0)]
        LOG.info(f"After filtering skipme: {len(original_dps)} data points")

        # Extract original and translated segment pairs
        evaluation_data_points = []
        dp_info_for_evaluation = []
        segment_boundaries = [0]  # Track where each document's segments start/end

        for dp in original_dps:
            if self.cfg.translation_key not in dp:
                LOG.warning(f"Data point missing '{self.cfg.translation_key}' key, skipping")
                dp_info_for_evaluation.append([])
                segment_boundaries.append(len(evaluation_data_points))
                continue

            translation_data = dp[self.cfg.translation_key]
            if "segmented_translation" not in translation_data:
                LOG.warning(f"Translation data missing 'segmented_translation' key, skipping")
                dp_info_for_evaluation.append([])
                segment_boundaries.append(len(evaluation_data_points))
                continue

            current_dp_info = []
            segmented_translations_map = translation_data["segmented_translation"]

            for field_path in self.cfg.fields_to_evaluate:
                has_wildcard = "*" in field_path
                path_parts = field_path.split(".")
                field_key = path_parts[-1]

                # Get segmented translations for this field
                if field_key in segmented_translations_map:
                    segmented_data = segmented_translations_map[field_key]
                    
                    # Handle both single document and multiple documents (for wildcard fields)
                    if isinstance(segmented_data, list):
                        if segmented_data and isinstance(segmented_data[0], dict) and "src" in segmented_data[0]:
                            # Single document with segments: [{"src": "...", "tgt": "..."}, ...]
                            segments_list = [segmented_data]
                        else:
                            # Multiple documents: [[{"src": "...", "tgt": "..."}, ...], [...]]
                            segments_list = segmented_data
                    else:
                        LOG.warning(f"Unexpected format for segmented_translation['{field_key}'], skipping")
                        continue

                    # Process each document's segments
                    for segments in segments_list:
                        if not segments:
                            continue
                        
                        current_dp_info.append((field_key, has_wildcard, len(segments)))
                        
                        # Create evaluation pairs for each segment
                        for segment in segments:
                            if isinstance(segment, dict) and "src" in segment and "tgt" in segment:
                                eval_data_point = {
                                    "source_text": segment["src"],
                                    "translated_text": segment["tgt"],
                                    "source_lang": self.cfg.source_lang,
                                    "target_lang": self.cfg.target_lang,
                                    "source_language": full_language_name(self.cfg.source_lang),
                                    "target_language": full_language_name(self.cfg.target_lang),
                                    self.cfg.async_position_key: len(evaluation_data_points),
                                }
                                evaluation_data_points.append(eval_data_point)
                            else:
                                LOG.warning(f"Segment missing 'src' or 'tgt' key: {segment}")
                else:
                    LOG.warning(f"Field '{field_key}' not found in segmented_translation, skipping")
                    continue

            dp_info_for_evaluation.append(current_dp_info)
            segment_boundaries.append(len(evaluation_data_points))

        # Store reconstruction data for postprocessing
        self._reconstruction_data = {
            "original_dps": original_dps,
            "dp_info_for_evaluation": dp_info_for_evaluation,
            "segment_boundaries": segment_boundaries,
        }

        LOG.info(f"Created {len(evaluation_data_points)} segment evaluation tasks from {len(original_dps)} documents")
        return evaluation_data_points

    def log_example_prompt(self, data):
        """Override log_example_prompt to work with evaluation data format."""
        if data:
            sample_data_point = (
                data[0]
                if isinstance(data[0], dict) and "source_text" in data[0]
                else {
                    "source_text": "Sample source text",
                    "translated_text": "Sample translated text",
                    "source_lang": self.cfg.source_lang,
                    "target_lang": self.cfg.target_lang,
                    "source_language": full_language_name(self.cfg.source_lang),
                    "target_language": full_language_name(self.cfg.target_lang),
                }
            )
            filled_prompt = self.fill_prompt(sample_data_point, data)
            LOG.info("Example prompt:\nData dictionary: %s\nPrompt: %s", sample_data_point, filled_prompt)
        else:
            LOG.info("No data available for example prompt")

    def generate(self):
        Path(self.cfg.output_file).absolute().parent.mkdir(parents=True, exist_ok=True)

        data = self.load_data()

        if len(data) == 0:
            LOG.info("No data to process, skipping generation")
        else:
            data = self.preprocess_data(data)
            data = self.skip_completed_samples(data)

            self.log_example_prompt(data)

            if self.cfg.dry_run:
                LOG.info("Exiting without running generation as dry_run flag is set.")
                return

            if not self.cfg.skip_filled:
                for output_path in [Path(self.cfg.output_file), Path(self.cfg.output_file + "-async")]:
                    if output_path.exists():
                        output_path.unlink()

            self.wait_for_server()
            asyncio.run(self.async_loop(data))

        if self.should_run_evaluation and self.evaluator is None:
            self.run_batch_evaluation()
        self.postprocess()

    def extract_scores_from_json(self, text: str) -> dict:
        """
        Extract FAITH scores from the generated JSON output.
        
        Expected format:
        {
          "Fluency": score,
          "Accuracy": score,
          "Idiomaticity": score,
          "Terminology": score,
          "Handling_of_Format": score
        }
        """
        # Try to find JSON in the text
        json_match = re.search(r'\{[^}]*\}', text, re.DOTALL)
        if json_match:
            try:
                scores_dict = json.loads(json_match.group(0))
                # Normalize the scores to expected keys
                normalized_scores = {}
                for key in ["Fluency", "Accuracy", "Idiomaticity", "Terminology", "Handling_of_Format"]:
                    if key in scores_dict:
                        normalized_scores[key] = float(scores_dict[key])
                    else:
                        # Try alternative key names
                        alt_key = key.replace("_", " ").replace("of", "of")
                        if alt_key in scores_dict:
                            normalized_scores[key] = float(scores_dict[alt_key])
                        else:
                            normalized_scores[key] = 0.0
                            LOG.warning(f"Score for '{key}' not found in response, setting to 0.0")
                return normalized_scores
            except (json.JSONDecodeError, ValueError) as e:
                LOG.error(f"Failed to parse JSON scores: {e}, text: {text}")
                return {
                    "Fluency": 0.0,
                    "Accuracy": 0.0,
                    "Idiomaticity": 0.0,
                    "Terminology": 0.0,
                    "Handling_of_Format": 0.0,
                }
        else:
            LOG.error(f"No JSON found in response: {text}")
            return {
                "Fluency": 0.0,
                "Accuracy": 0.0,
                "Idiomaticity": 0.0,
                "Terminology": 0.0,
                "Handling_of_Format": 0.0,
            }

    def average_scores(self, scores_list: List[dict]) -> dict:
        """
        Average FAITH scores across multiple segments, rounded to 2 decimal places.
        
        Args:
            scores_list: List of score dictionaries
            
        Returns:
            Dictionary with averaged scores (rounded to 2 decimal places)
        """
        if not scores_list:
            return {
                "Fluency": 0.0,
                "Accuracy": 0.0,
                "Idiomaticity": 0.0,
                "Terminology": 0.0,
                "Handling_of_Format": 0.0,
            }
        
        averaged = {}
        for key in ["Fluency", "Accuracy", "Idiomaticity", "Terminology", "Handling_of_Format"]:
            values = [score[key] for score in scores_list if key in score]
            averaged[key] = round(sum(values) / len(values), 2) if values else 0.0
        
        return averaged

    def reconstruct_messages_with_translations(self, original_messages, translated_contents, field_path):
        """
        Reconstruct messages array with translated content.
        
        Args:
            original_messages: Original messages array or nested structure
            translated_contents: List of translated content strings
            field_path: Field path like "conversations.*.value" or "messages.*.content"
            
        Returns:
            Tuple of (reconstructed structure, number of translations used)
        """
        path_parts = field_path.split(".")
        
        # If this is messages.*.content pattern
        if len(path_parts) == 3 and path_parts[0] == "messages" and path_parts[1] == "*":
            content_field = path_parts[2]  # e.g., "content"
            
            if not isinstance(original_messages, list):
                return original_messages, 0
            
            new_messages = []
            translation_idx = 0
            
            for msg in original_messages:
                if isinstance(msg, dict):
                    new_msg = msg.copy()
                    if (content_field in msg and 
                        isinstance(msg[content_field], str) and 
                        translation_idx < len(translated_contents)):
                        new_msg[content_field] = translated_contents[translation_idx]
                        translation_idx += 1
                    new_messages.append(new_msg)
                else:
                    new_messages.append(msg)
            
            return new_messages, translation_idx
        
        # For other patterns like conversations.*.value, return None to use default handling
        return None, 0

    def postprocess(self):
        """
        Add FAITH scores back to the original data points.
        Averages scores across segments for each document.
        Reads the evaluation results written by base class and merges scores.
        
        Additionally, reconstructs messages with translated content:
        - Original messages are stored in 'original_messages'
        - Translated messages are stored in 'messages'
        """

        LOG.info("Starting postprocessing: adding FAITH scores to original data")

        # Read the evaluation results written by base class
        output_file = Path(self.cfg.output_file)
        if not output_file.exists():
            LOG.warning(f"Output file {output_file} does not exist, skipping postprocessing")
            return

        # Load evaluation results
        evaluation_results = []
        with open(output_file, "r", encoding="utf-8") as f:
            for line in f:
                if line.strip():
                    evaluation_results.append(json.loads(line))

        LOG.info(f"Loaded {len(evaluation_results)} segment evaluation results")

        # Extract scores from results
        all_segment_scores = []
        for result in evaluation_results:
            # The base class should have stored the generation in 'generation' field
            if "generation" in result:
                score_text = result["generation"]
            elif "response" in result:
                score_text = result["response"]
            else:
                score_text = str(result)
            
            scores = self.extract_scores_from_json(score_text)
            all_segment_scores.append(scores)

        reconstruction_data = self._reconstruction_data
        original_dps = reconstruction_data["original_dps"]
        dp_info_for_evaluation = reconstruction_data["dp_info_for_evaluation"]
        segment_boundaries = reconstruction_data["segment_boundaries"]

        # Add scores back to original data points
        final_output_data = []
        
        for i, dp in enumerate(original_dps):
            output_dp = dp.copy()
            dp_info = dp_info_for_evaluation[i]
            
            if not dp_info:
                final_output_data.append(output_dp)
                continue

            # Get all segment scores for this data point
            start_idx = segment_boundaries[i]
            end_idx = segment_boundaries[i + 1]
            dp_segment_scores = all_segment_scores[start_idx:end_idx]

            if not dp_segment_scores:
                final_output_data.append(output_dp)
                continue

            # Group scores by field (for documents with multiple fields)
            field_scores_map = {}
            segment_idx = 0
            
            for field_key, has_wildcard, num_segments in dp_info:
                # Collect scores for this field's segments
                field_segment_scores = dp_segment_scores[segment_idx:segment_idx + num_segments]
                segment_idx += num_segments
                
                # Average scores across segments
                averaged_scores = self.average_scores(field_segment_scores)
                
                # Store both individual segment scores and averaged score
                if has_wildcard:
                    if field_key not in field_scores_map:
                        field_scores_map[field_key] = []
                    field_scores_map[field_key].append({
                        "averaged_scores": averaged_scores,
                        "segment_scores": field_segment_scores,
                        "num_segments": num_segments
                    })
                else:
                    field_scores_map[field_key] = {
                        "averaged_scores": averaged_scores,
                        "segment_scores": field_segment_scores,
                        "num_segments": num_segments
                    }

            # Add scores to output
            output_dp[self.cfg.scores_key] = {
                "source_lang": self.cfg.source_lang,
                "target_lang": self.cfg.target_lang,
                "field_scores": field_scores_map,
            }
            
            # Reconstruct messages with translations if translations exist and messages field is evaluated
            if self.cfg.translation_key in dp and "messages" in dp:
                translation_data = dp[self.cfg.translation_key]
                
                # Check if we're evaluating messages (look for content or value field)
                for field_path in self.cfg.fields_to_evaluate:
                    path_parts = field_path.split(".")
                    if len(path_parts) >= 1 and path_parts[0] == "messages":
                        # Get the field key (e.g., "content" from "messages.*.content")
                        field_key = path_parts[-1] if len(path_parts) > 1 else "content"
                        
                        # Get translated content from translation data
                        if "translation" in translation_data and field_key in translation_data["translation"]:
                            translated_contents = translation_data["translation"][field_key]
                            
                            # Handle both single string and list of strings
                            if isinstance(translated_contents, str):
                                translated_contents = [translated_contents]
                            elif not isinstance(translated_contents, list):
                                continue
                            
                            # Store original messages
                            output_dp["original_messages"] = dp["messages"]
                            
                            # Reconstruct messages with translations
                            reconstructed, _ = self.reconstruct_messages_with_translations(
                                dp["messages"], translated_contents, field_path
                            )
                            
                            if reconstructed is not None:
                                output_dp["messages"] = reconstructed
                            
                            break  # Only process once per data point
            
            final_output_data.append(output_dp)

        # Write results back to output file
        LOG.info(f"Writing {len(final_output_data)} documents with FAITH scores to {output_file}")
        with open(output_file, "w", encoding="utf-8") as f:
            for dp in final_output_data:
                f.write(json.dumps(dp, ensure_ascii=False) + "\n")

        LOG.info("Postprocessing completed successfully")

"""
Abstract base class for translation backends.

All translation backends (NMT, Google, AWS) must implement this interface.
Common functionality like async loop, restore order, and translate flow
are implemented here to avoid duplication.
"""

import asyncio
import json
import logging
import time
from abc import ABC, abstractmethod
from pathlib import Path
from typing import List
from tqdm import tqdm

LOG = logging.getLogger(__name__)


class TranslationBackend(ABC):
    """
    Abstract base class for translation backends.
    
    Provides common functionality:
    - translate(): Template method for the main translation flow
    - _async_loop(): Async loop with semaphore-based concurrency
    - _restore_order(): Restore async results to original order
    
    Subclasses must implement:
    - _backend_name: Property returning backend name for logging
    - _setup_client(): Initialize backend-specific client
    - check_server(): Verify backend service is available
    - _translate_single_async(): Translate a single text asynchronously
    """
    
    def __init__(self, cfg: "TranslationConfig"):
        """
        Initialize backend with config.
        
        Args:
            cfg: TranslationConfig
        """
        self.cfg = cfg
        # Async infrastructure - common to all backends
        self._semaphore = asyncio.Semaphore(cfg.max_concurrent_requests)
        self._output_lock = None  # Initialized in async context
    
    @property
    @abstractmethod
    def _backend_name(self) -> str:
        """Return backend name for logging (e.g., 'Google', 'AWS', 'NMT')."""
        pass
    
    @abstractmethod
    def _setup_client(self) -> None:
        """Initialize backend-specific client/connection."""
        pass
    
    @abstractmethod
    def check_server(self) -> bool:
        """
        Check if the translation server/service is available.
        
        Returns:
            True if backend is reachable/healthy, False otherwise.
        """
        pass
    
    @abstractmethod
    async def _translate_single_async(self, text: str) -> str:
        """
        Translate a single text asynchronously.
        
        This is the core translation logic that each backend must implement.
        
        Args:
            text: Source text to translate
            
        Returns:
            Translated text
        """
        pass
    
    def close(self) -> None:
        """
        Cleanup resources (e.g., close HTTP sessions).
        
        Override if backend needs cleanup.
        """
        pass
    
    def translate(self, data: List[dict]) -> None:
        """
        Translate preprocessed data and write results to output file.

        Args:
            data: List of preprocessed data points with "src" and "_async_position" keys
        """
        LOG.info(
            f"{self._backend_name} Translation: max_concurrent={self.cfg.max_concurrent_requests}, "
            f"total_texts={len(data)}"
        )
        
        if self.cfg.dry_run:
            LOG.info("Dry run mode - exiting without generation")
            return
        
        # Clean up existing output files
        if not self.cfg.skip_filled:
            for output_path in [Path(self.cfg.output_file), Path(self.cfg.output_file + "-async")]:
                if output_path.exists():
                    output_path.unlink()
        
        ok = self.check_server()
        if not ok:
            raise RuntimeError(
                f"{self._backend_name} backend health check failed; aborting translation."
            )
        
        # Run async translation loop
        asyncio.run(self._async_loop(data))
        
        LOG.info(f"{self._backend_name} translation completed")

    async def _async_loop(self, data: list):
        """
        Async loop using semaphores for concurrency control.
        
        Creates one task per data point, uses semaphore to limit concurrent
        requests, and writes results in streaming fashion.
        """
        # Initialize output lock for thread-safe writing
        if self._output_lock is None:
            self._output_lock = asyncio.Lock()
        
        pbar = tqdm(total=len(data), desc=f"{self._backend_name} translations")
        
        with open(self.cfg.output_file + "-async", "at", encoding="utf-8", buffering=1) as fout:
            # Create tasks for all data points
            tasks = [
                asyncio.create_task(self._translate_and_save(dp, fout, pbar))
                for dp in data
            ]
            
            # Wait for all tasks to complete
            if tasks:
                await asyncio.gather(*tasks)
        
        pbar.close()
        
        # Restore original order
        self._restore_order()
    
    async def _translate_and_save(self, data_point: dict, fout, pbar):
        """
        Translate a single data point and save result.
        
        Uses semaphore to limit concurrent requests and output_lock
        for thread-safe file writing.
        """
        text = data_point.get("src", "")
        
        # Use semaphore to limit concurrent requests
        async with self._semaphore:
            start_time = time.time()
            translation = await self._translate_single_async(text)
            end_time = time.time()
        
        # Build result
        result = {
            **data_point,
            "generation": translation,
            "generation_start_time": start_time,
            "generation_end_time": end_time,
            "generation_time": end_time - start_time,
        }
        
        # Thread-safe output writing
        async with self._output_lock:
            fout.write(json.dumps(result, ensure_ascii=False) + "\n")
            pbar.update(1)
    
    def _restore_order(self):
        """
        Restore original order of results after async processing.
        
        Reads the async output file, reorders by _async_position key,
        and writes to the final output file.
        """
        async_file = Path(self.cfg.output_file + "-async")
        if not async_file.exists():
            LOG.warning("No async file found, skipping order restoration")
            return
        
        with open(async_file, "rt", encoding="utf-8") as fin:
            generations = [json.loads(line) for line in fin if line.strip()]
        
        ordered_generations = [None] * len(generations)
        for gen_dict in generations:
            async_pos = gen_dict.pop(self.cfg.async_position_key)
            ordered_generations[async_pos] = gen_dict
        
        with open(self.cfg.output_file, "wt", encoding="utf-8") as fout:
            for gen_dict in ordered_generations:
                if gen_dict is not None:
                    fout.write(json.dumps(gen_dict, ensure_ascii=False) + "\n")
        
        async_file.unlink()
        LOG.info(f"Restored order for {len(generations)} translations")
